a console battleship game written in Java

## How to use it:
- Clone the repository to your local machine
  - $ `git clone https://github.com/thedtripp/battleship.git`
  - $ `cd battleship`
- Compile the Java files
  - $ `javac *.java`
- Run the main Java file
  - $ `java BattleShipMain`
- Sink some battleships
